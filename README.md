# Survey Form: Software Engineer
Ini adalah survey form yang saya buat untuk certification project Responsive Web Design. Saya hanya menggunakan HTML dan CSS untuk membuat form ini. Saya mengambil topik software engineer karena saya tertarik dengan bidang ini.

## Tujuan
Tujuan dari survey form ini adalah untuk mengetahui minat, pengalaman, dan preferensi para software engineer di Indonesia. Saya ingin mengetahui apa yang membuat mereka tertarik dengan bidang ini, apa yang mereka pelajari, dan apa yang mereka harapkan dari karir mereka.

## Isi Form
Form ini terdiri dari beberapa pertanyaan yang berkaitan dengan software engineer, yaitu:

- Nama (input text): Nama lengkap Anda.
- Email (input text): Alamat email Anda yang valid.
- Pengalaman (input number): Jumlah tahun Anda bekerja sebagai software engineer.
- Bidang Spesialis (input select): Bidang spesialis Anda sebagai software engineer, misalnya web development, mobile development, data science, dll.
- Bahasa Favorit (input radio): Bahasa pemrograman favorit Anda, misalnya Java, Python, C#, dll.
- Framework yang Ingin Dipelajari (input checkbox): Framework yang ingin Anda pelajari atau kuasai lebih lanjut, misalnya React, Angular, Django, dll.
- Komentar (textarea): Komentar atau saran Anda tentang survey form ini atau tentang software engineer secara umum.
- Submit (input submit): Tombol untuk mengirimkan jawaban Anda.
